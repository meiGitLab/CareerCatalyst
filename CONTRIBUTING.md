# Contributing to CareerCatalyst

I welcome contributions to CareerCatalyst! This project originated from a training and I'm excited to see it grow with community input.

## Ways to Contribute

1. **Report Bugs**: Open an issue describing the bug and how to reproduce it
2. **Suggest Features**: Share your ideas for new features or improvements
3. **Code Contributions**: Submit pull requests for bug fixes or new features
4. **Share Feedback**: Tell me about your experience using CareerCatalyst

## Development Setup

1. Fork the repository
2. Create a virtual environment
3. Install dependencies: `pip install -r requirements.txt`
4. Ensure Ollama is running locally
5. Make your changes
6. Submit a pull request

## Code Style

- Follow PEP 8 guidelines
- Include docstrings for functions and classes
- Add comments for complex logic
- Update documentation as needed
